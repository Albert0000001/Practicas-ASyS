Problema 3

Para este problema se identificaron las fórmulas por las que se calcula tanto la potencia como la energía y de esta forma se llegaron a las siguientes funciones:
Función para la energía:
function Ex = Ene(x)
if ~isvector(x)
error('Debe ser un vector')
end
Ex = sum(x.^2);
end
Función para la potencia de una señal periódica:
function Px = Pot(x)
if ~isvector(x)
error('Debe ser un vector')
end
Px = (1/length(x))*sum(x.^2);
end
